# BeagleBoneOpenBoard
BeagleBone Open Board
Mạch hỗ trợ học linux bằng beagle bone black.
Mạch hỗ trợ:
	- GPIO, interrupt: nút nhấn và led
	- I2c: SSD1306
	- SPI: ILI9341, Nokia5110
Muốn debug thì vẫn bắt buộc phải hàn vào các pin hoặc cắm trên broadboard
